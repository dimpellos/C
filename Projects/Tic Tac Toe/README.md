# Tic-Tac-Toe Console Game

This is a simple Tic-Tac-Toe game written in C for the console.

## How to Play

- The game board is always displayed on the console.
- Move the cursor using the following keys (either lowercase or uppercase):  
  - **W**: Move cursor up  
  - **A**: Move cursor left  
  - **S**: Move cursor down  
  - **D**: Move cursor right  
- When you want to place your move on the selected spot, press **P**.

Enjoy playing Tic-Tac-Toe!
